1부: Vibe Coding(바이브 코딩)

대규모 데이터에서 패턴을 학습해 프롬프트에 따라 새로운 텍스트·이미지·오디오·영상·코드 등을 생성·변환하는 AI입니다.

텍스트 생성

글쓰기, 번역, 요약

이미지 생성

일러스트, 사진, 디자인

코드 생성

프로그래밍, 웹 개발

AI(LLM)에 자연어로 명령(프롬프트)을 입력하여 코드를 생성하는 새로운 프로그래밍 방식입니다. 개발자가 원하는 기능, '분위기(Vibe)'를 설명하면 AI가 그에 맞춰 코드를 작성해주는 것이 특징입니다.

  • 자연어 기반: 복잡한 코딩 지식 없이, 원하는 의도를 AI에게 전달하면 코드로 전환됩니다.
  • 협업 중심: 개발자가 AI와 대화하고, 코드를 테스트/수정하는 역할로 변화합니다.
  • 접근성 증가: 프로그래밍 초보자도 앱 개발이 가능해져 개발 문턱이 낮아집니다.
  • 속도와 창의성: 개발자의 직관적인 아이디어가 바로 구현되며 창의적인 결과물이 빠르게 나옵니다.

장점 (Pros)

  • 누구나 빠르게 앱/웹서비스 개발 가능
  • 프로토타이핑 속도 증가
  • 개발자의 역할 확장 (기획/검토)

! 한계 (Cons)

  • AI가 만든 코드는 유지보수/디버깅이 어려울 수 있습니다.
  • 최종 검토와 수정은 여전히 사람의 몫으로 매우 중요합니다.

AI에게 명확하게 요청하는 것이 중요합니다.

  • 명확한 역할 지정
  • 구체적인 요청
  • 형식 지정
  • 예시 제공

좋은 프롬프트 vs 나쁜 프롬프트

X 나쁜 예시

"수업 자료 만들어줘"

너무 모호하고 구체적인 정보가 없습니다.

✔ 좋은 예시

"중학교 1학년 수학 시간에 사용할 일차방정식 개념 설명 자료를 만들어줘. 예시 3개와 연습문제 5개를 포함해서 PPT 슬라이드 3장 분량으로 작성해줘."

대상, 과목, 주제, 형식, 분량이 모두 명확합니다.
